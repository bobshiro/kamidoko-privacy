# 紙マップでどこ？ プライバシーポリシー

**適用日**: 2026年6月7日
**最終更新日**: 2026年6月7日

紙マップでどこ？（以下「本アプリ」といいます）は、ユーザーの皆様のプライバシーを尊重し、個人情報の取り扱いに最大限の注意を払っています。本ポリシーは、本アプリがどのような情報を扱い、どのように利用するかを説明するものです。

本アプリは、紙の案内図・手描きの地図・絵地図などの上に、お使いの端末のGPS現在地を重ねて表示するアプリです。地図を友だちと共有して、お互いの位置をリアルタイムで見せ合うこともできます。

## 2. 本アプリが端末内に保存する情報

本アプリは、ユーザーが作成した以下の情報を **端末内** に保存します。これらは地図の表示・位置合わせという利用目的のために使われ、ユーザーが「マップ共有」を行わない限り、開発者を含む第三者のサーバには送信されません。

- 取り込んだ地図の画像（写真・カメラ撮影・PDF・絵地図など）
- 位置合わせ（キャリブレーション）の地点：地図上で指定した点と、それに対応するGPS座標
- 地図上に置いたピンやメモ
- 表示名（マップ共有時に相手の画面へ表示される名前）
- アプリの設定

これらのデータは、ユーザーが本アプリを削除した時点で端末から消去されます。

---

## 3. 位置情報の取り扱い

本アプリは、現在地を地図上に表示するために位置情報を利用します。

- 取得した位置情報は、通常は **端末内のみ** で利用し、保存や送信は行いません。
- 後述の「マップ共有」を有効にしている間に限り、現在地を共有相手の端末へ送信します。このときは、アプリを閉じていても位置を送り続けられるよう、**バックグラウンドでも位置情報を利用** します。
- マップ共有を停止すると、位置情報の送信は止まります。

---

## 4. マップ共有と iCloud（第三者への送信について）

本アプリの中で情報が端末の外へ送信されるのは、**ユーザー自身が「マップ共有」を行ったときだけ** です。

共有を開始すると、Apple の iCloud（CloudKit）を通じて、以下の情報が共有相手の端末へ送信されます。

- あなたの **現在地（精密な位置情報）**
- 地図上の **位置合わせ地点・ピン**
- あなたの **表示名**

ご利用にあたっては、以下の点に特にご注意ください。

- 共有リンク（URL）を知っている人は、**誰でも** その地図とリアルタイムの位置を見ることができます。共有は、信頼できる相手にのみ行ってください。
- マップ共有を停止すると、リアルタイムの位置情報の送信は止まります。
- これらのデータは Apple の iCloud 基盤を通じてやり取りされ、**開発者がアクセスできる形では保存されません**。

## 5. 端末の権限利用について

本アプリは、機能の提供のため以下の権限を、ユーザーの許可のもとで利用します。

| 権限 | 利用目的 |
| --- | --- |
| カメラ | 紙の地図や案内図をその場で撮影し、新しいマップを作成するため |
| 写真ライブラリ | 端末内の写真を選んでマップの元画像にするため（写真ピッカー経由のため、写真ライブラリ全体へのアクセス許可は求めません） |
| 位置情報（使用中のみ） | 現在地を地図上に表示するため |
| 位置情報（常に許可／バックグラウンド） | マップ共有中に、アプリを閉じていても自分の位置を相手の端末へ送り続けるため |

これらの権限は **ユーザーが拒否しても本アプリの基本機能は動作** します（該当する機能のみ利用できなくなります）。

## 6. サブスクリプション（課金）について

本アプリは、一部の機能を自動更新のサブスクリプション（月額・無料トライアルつき）として提供しています。

- 決済は **Apple（App Store）を通じて** 行われ、クレジットカード番号などの支払い情報が開発者に渡ることはありません。
- 課金の対象は地図を **作成する人** のみで、共有された地図を受け取って閲覧・利用する相手に課金は発生しません。
- サブスクリプションの確認・解約は、いつでも端末の「設定」＞ご自身の Apple Account ＞「サブスクリプション」から行えます。
- 詳しくは [Apple のプライバシーポリシー](https://www.apple.com/legal/privacy/) もご確認ください。

## 7. 利用する第三者サービス

本アプリが利用する外部サービスは、Apple が提供する以下のものに限られます。

- **Apple iCloud（CloudKit）**: マップの共有・同期のため（前述の「マップ共有」を行った場合のみ）
- **Apple App Store**: サブスクリプションの決済・管理のため

本アプリは、上記以外の第三者サービスを利用していません。具体的には、**広告の配信、アクセス解析（アナリティクス）、ユーザーを追跡（トラッキング）する仕組みは一切ありません**。Google などの外部の地図・検索 API も利用していません。

---

## 8. データの保管・セキュリティ

- 端末内に保存されるデータには、iOS 標準のファイル保護（File Protection）が適用されます。
- iCloud 同期やマップ共有で扱われるデータは、Apple の iCloud 基盤を通じてやり取りされ、開発者がアクセスできる形では保存されません。
- 地図をファイル（.gpsmap 形式）として書き出した場合、その保存場所はユーザーの管理下にあります。書き出したファイルには位置情報が含まれることがあるため、SNS への投稿など不特定多数への共有はお控えください。
